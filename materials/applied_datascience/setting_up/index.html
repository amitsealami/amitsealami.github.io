Setting up Work Environment

Thu, Jun 22, 2017 datascience, course

We will be using the Cloudera-Udacity Hadoop course materials.

  1. Head over udacity if you have not registered already.
  2. Next, download the material from here called “Cloudera-Udacity-Training-VM-4.1.1.c.zip”. It is around 2GB in size, so make sure you download it once and then share it amongst other classmates.

  3. Additionally, install 7zip if you do not have already in Windows. It is considerably faster for unzipping materials. If you are in Linux - you need to open terminal and type

    unzip Cloudera-Udacity-Training-VM-4.1.1.c.zip -d udacity-cloudera

  4. There are several files related to lightweight Hadoop setup from Cloudera in virtual disk image format. Even though it is possible to use VirtualBox to host the virtual image, VMware is the preferred choice since it performs better.

  5. If you are in Linux, you need to download the bundle file. When we downloaded it, it was VMware-Player-12.5.6-5528349.x86_64.bundle. You need to execute the following from terminal:

    chmod +x VMware-Player-12.5.6-5528349.x86_64.bundle
sudo ./VMware-Player-12.5.6-5528349.x86_64.bundle

    When asked whether to install, install it.

  6. Then, open VM Player. Click on Open a virtual machine, and select the vmx file from the extracted directory!
